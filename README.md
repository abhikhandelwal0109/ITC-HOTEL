# ITC-HOTEL

1.Project Title / Headline
Hospitality Analytics Dashboard – ITC Hotels

2.Short Description / Purpose
This project presents an interactive Power BI dashboard designed to analyze hotel performance and booking trends for ITC Hotels. The dashboard helps users understand revenue patterns, occupancy rate, and booking behavior to support better business decisions.

3.Tech Stack

.Power BI Destop –Data Visualization Platform Used For Creation Report
.Power Query-Data Transformation and Cleaning and Preparing The Data
.Dax(Data Analysis Expression)-Calculated Measures,Dynamic Visuals,Conditional Logis
.Data Modeling-Relationship Between Among Tables


4.Data Source
The dataset used in this project contains hotel booking and revenue data, including information about bookings, check-ins, room categories, occupancy rate, and revenue generated by different hotels.

5.Features / Highlights
 Business Problem
 Hotel management needs a clear view of booking performance, revenue generation, and occupancy trends to improve operational efficiency.
 Goal of the Dashboard
 To create an interactive Power BI dashboard that helps analyze hotel booking data and monitor key performance metrics.
 Walkthrough of Key Visuals
 Revenue by Hotel – Shows which hotel generates the most revenue.
 Booking Trends – Displays booking patterns over time.
 Occupancy Rate Analysis – Helps track how effectively hotel rooms are being utilized.
 Room Category Performance – Compares bookings across different room types.
 Highlight the Insights
 Identifies top-performing hotels.
 Shows peak booking periods.
 Helps understand customer booking trends.
 Business Impact
 This dashboard helps management make data-driven decisions, improve occupancy rates, and optimize revenue performance.
